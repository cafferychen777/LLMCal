Get Started

  1. Download the latest release: Releases
  2. Double‑click the LLMCal.popclipext.zip file to install
  3. Open PopClip → Options → LLMCal → Enter your Anthropic API key → Save
  4. Select text describing an event → Click the calendar icon

First Event Example

"Team meeting tomorrow at 2pm for 1 hour, Zoom https://zoom.us/j/123, remind me 15 minutes before"

Troubleshooting

  • Ensure PopClip has Accessibility and Calendar permissions
  • Check your API key is valid
  • See INSTALLATION and TROUBLESHOOTING
